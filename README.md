# Browser-Technology
Mijn repo voor het vak Browser Technology


## Mijn leerdoelen
- Ik wil leren hoe ik creatieve en speelse interacties kan bouwen met HTML, CSS en JavaScript in plaats van “alleen functionele” interfaces.
- Ik wil beter worden in experimenteren, niet terughoudend zijn met dingen uitproberen.
- Ik wil beter worden in het maken en vastleggen van verschillende iteraties en hoe feedback hierin terugkomt.

**Hoeveel tijd heeft me dat gekost?**

    Elementen liggen niet altijd voor de hand. Bijvoorbeeld bij het invullen van de datum bij 1b is het niet handig om een date te gebruiken, omdat het kan zijn dat je veel jaren terug moet.     
**Wat ga ik morgen doen?**  
    Verder met vraag 1. Ik wil de html structuur zo goed als af hebben zodat ik daarna kan kijken naar hoe je vragen verbergt/overslaat.
